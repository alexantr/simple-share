Simple Share

Simple Share Buttons for Your Site

GitHub Download simple-share.js

Current version: 2.0

Examples

Example #1 - Basic usage #

Note: Only Facebook, VKontakte and Twitter are enabled by default.

HTML:

<script src="https://code.jquery.com/jquery-3.3.1.min.js"></script>
<script src="js/simple-share.min.js"></script>

HTML:

<div class="simple-share example-1"></div>

JavaScript:

$(".simple-share.example-1").simpleShare();

Result:

Example #2 - Sorting and disabling #

HTML:

<div class="simple-share example-2"></div>

JavaScript:

$(".simple-share.example-2").simpleShare({
    enabledProviders: ["vkontakte", "facebook", "linkedin", "telegram"]
});

Result:

Example #3 - All providers #

HTML:

<div class="simple-share example-3"></div>

JavaScript:

$(".simple-share.example-3").simpleShare({
    enabledProviders: "*"
});

Result:

Using data-* attributes #

Use data-* attributes data-url, data-title and data-image:

<div class="simple-share" data-url="http://example.com/" data-title="Example Title" data-image="http://example.com/image.jpg"></div>

Note: data-image is required for Pinterest:

HTML:

<div class="simple-share example-5" data-image="https://yashkin.by/simple-share/images/simple-share.png"></div>

JavaScript:

$(".simple-share.example-5").simpleShare({
    enabledProviders: ["pinterest"]
});

Result:

Providers in data-enabled-providers #

Use data-enabled-providers with providers separated by comma:

HTML:

<div class="simple-share example-6" data-enabled-providers="facebook,vkontakte,linkedin"></div>
<div class="simple-share example-6" data-enabled-providers="*"></div>

JavaScript:

$(".simple-share.example-6").simpleShare();

Result:
