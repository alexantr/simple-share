Simple Share

Simple Share Buttons for Your Site

Download simple-share.min.js GitHub

Examples

Basic usage #

HTML:

<script src="https://code.jquery.com/jquery-3.4.1.min.js"></script>
<script src="simple-share.min.js"></script>
<div class="simple-share"></div>

Result:

Note: only Facebook, VKontakte and Telegram are enabled by default.

Configure providers #

HTML:

<div class="simple-share" data-providers="facebook,linkedin,telegram,viber,whatsapp,skype"></div>

Enable all providers with *

<div class="simple-share" data-providers="*"></div>

Result:

Additional attributes #

Use attributes data-url, data-title and data-image to configure share message:

<div class="simple-share" data-url="http://example.com/" data-title="Example Title" data-image="http://example.com/image.jpg"></div>

By default plugin uses title from "og:title" or document.title, url from window.location.href and image from "og:image".

Note: data-image is required for Pinterest if meta "og:image" is not set.

HTML:

<div class="simple-share" data-providers="pinterest" data-image="https://tmpfiles.xyz/simple-share/images/simple-share.png"></div>

Result:

Manual initialization and plugin options #

<p class="share-custom"></p>
<p class="share-custom" data-providers="viber,telegram"></p>

Note: data-* attributes have higher priority.

JavaScript:

$('.share-custom').simpleShare({
    providers: 'vkontakte,facebook,pinterest',
    styled: false,
    image: 'https://tmpfiles.xyz/simple-share/images/simple-share.png',
    url: 'https://tmpfiles.xyz/simple-share/',
    title: 'Simple Share'
}).find('a').each(function () {
    $(this).css({color: $(this).attr('data-color')});
});

Note: option "styled" with false disables built-in CSS. You must set own styles.

CSS:

.share-custom a {
    display: inline-block;
    margin-right: 5px;
    transition: all .2s;
}
.share-custom a:hover,
.share-custom a:focus {
    opacity: 0.8;
}
.share-custom svg {
    width: 2rem;
    height: 2rem;
}

Result:

Manual initialization with default class #

Add data-skip-auto attribute to disabled auto initialization:

<div class="simple-share" data-skip-auto>Waiting...</div>

JavaScript:

$('#load-share').on('click', function () {
    $('.simple-share').simpleShare(); // also run if loading buttons throw AJAX
});
