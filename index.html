Simple Share 2

Simple Share Buttons for Your Site

GitHub Download simple-share.min.js

Current version: 2.0-dev

Examples

Basic usage #

HTML:

<script src="https://code.jquery.com/jquery-3.3.1.min.js"></script>
<script src="simple-share.min.js"></script>
<div class="simple-share2"></div>

Result:

Note: only Facebook, VKontakte and Twitter are enabled by default.

Configure providers #

HTML:

<div class="simple-share2" data-providers="facebook,linkedin,telegram,viber,whatsapp,skype"></div>

Result:

All providers #

HTML:

<div class="simple-share2" data-providers="*"></div>

Result:

Additional attributes #

Use attributes data-url, data-title and data-image to configure share message:

<div class="simple-share2" data-url="http://example.com/" data-title="Example Title" data-image="http://example.com/image.jpg"></div>

By default plugin uses title from "og:title" or document.title, url from window.location.href and image from "og:image".

Note: data-image is required for Pinterest if meta "og:image" is not set.

HTML:

<div class="simple-share2" data-providers="pinterest" data-image="https://yashkin.by/simple-share/images/simple-share.png"></div>

Result:

Manual initialization and plugin options #

<div class="example-custom"></div>
<div class="example-custom" data-providers="viber,telegram"></div>

Note: data-* attributes have higher priority.

JavaScript:

$('.example-custom').simpleShare2({
    providers: 'vkontakte,facebook,pinterest',
    styled: false,
    image: 'https://yashkin.by/simple-share/images/simple-share.png',
    url: 'https://yashkin.by/simple-share/',
    title: 'Simple Share 2'
});

Note: option "styled" with false disables built-in CSS. You must set own styles.

CSS:

.example-custom {
    margin-bottom: 10px;
}
.example-custom a {
    display: inline-block;
    margin-right: 5px;
    padding: 5px;
    border-radius: 50%;
}
.example-custom a:hover,
.example-custom a:focus {
    opacity: 0.9;
}
.example-custom img {
    width: 30px;
    height: 30px;
}
