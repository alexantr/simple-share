Simple Share

Simple Share Buttons for Your Site

GitHub Download simple-share.js

Current version: 2.0

Examples

Basic usage #

Note: Only Facebook, VKontakte and Twitter are enabled by default.

HTML:

<script src="https://code.jquery.com/jquery-3.3.1.min.js"></script>
<script src="simple-share.min.js"></script>

HTML:

<div class="simple-share example-basic"></div>

JavaScript:

$(".simple-share.example-basic").simpleShare();

Result:

Sorting and disabling #

HTML:

<div class="simple-share example-custom"></div>

JavaScript:

$(".simple-share.example-custom").simpleShare({
    enabledProviders: ["vkontakte", "facebook", "linkedin", "telegram"]
});

Result:

All providers #

HTML:

<div class="simple-share example-all"></div>

JavaScript:

$(".simple-share.example-all").simpleShare({
    enabledProviders: "*"
});

Result:

Using data-* attributes #

Use data-* attributes data-url, data-title and data-image:

<div class="simple-share" data-url="http://example.com/" data-title="Example Title" data-image="http://example.com/image.jpg"></div>

Note: data-image is required for Pinterest:

HTML:

<div class="simple-share example-pinterest" data-image="https://yashkin.by/simple-share/images/simple-share.png"></div>

JavaScript:

$(".simple-share.example-pinterest").simpleShare({
    enabledProviders: ["pinterest"]
});

Result:

Providers in data-enabled-providers #

Use data-enabled-providers with providers separated by comma:

HTML:

<div class="simple-share example-providers" data-enabled-providers="telegram,viber,whatsapp,skype"></div>
<div class="simple-share example-data" data-enabled-providers="*"></div>

JavaScript:

$(".simple-share.example-data").simpleShare();

Result:
